## Road Guardian 🚗🛡️

Road Guardian is a full-stack web application focused on road safety and monitoring. The platform provides real-time monitoring, alert mechanisms, and a modern web interface to help improve road safety awareness.

## Features

- Real-time monitoring system
- Secure backend API
- Responsive web interface
- Database integration with Supabase
- Alert and notification support
- Modern full-stack architecture

## Tech Stack

Frontend

- React
- Vite
- JavaScript
- HTML
- CSS

Backend

- Node.js
- Express.js

Database

- Supabase

## Project Structure

backend/
public/

## Installation

Clone Repository

git clone https://github.com/dhairyagemini/Road-Guardian.git
cd Road-Guardian

Install Frontend Dependencies

npm install

Install Backend Dependencies

cd backend
npm install

Run Frontend

npm run dev

The frontend will start on the Vite development server.

Run Backend

Open another terminal and run:

cd backend
node server.js

The backend API server will start locally.
